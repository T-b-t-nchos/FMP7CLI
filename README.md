# FMP7CLI
<sub>A simple command line interface for FMP7</sub>

## 概要 - Overview
<sub>FMP7CLI is a simple command line interface for FMP7.</sub>  
FMP7CLI は FMP7 のコマンドラインインターフェースです。  
<sub>It allows simple FMP7 operations.</sub>  
簡易的なFMP7の操作が可能です。  

## 使い方 - Usage

<sub>Please install the following software.</sub>  
下記のソフトウェアをインストールしてください。  
- FMP7
- .NET Framework 4.8.1 runtime

> [!TIP]
> ・<sub>You need to start FMP7 before using FMP7CLI.</sub>  
> ・FMP7 が起動している必要があります。

<sub>Please refer to the --help option.</sub>  
--help オプションを参照してください。
